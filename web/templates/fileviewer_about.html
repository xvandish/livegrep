This fileviewer is a rip-off of the publically Google fileviewer for cs.opensource.google.

Goals

  • Provide a fast, near local-disk speed experience while browsing a trees files
  • Provide access to common Git operations, log, blame, diff, for the current file
  • Make it easy to search within and browse files
  • Provide an intuitive browsing experience

Features

View Repo At Different Revisions

Repo Selector

This selector will allow you to switch between all the repositories that livegrep is configured to allow filebrowsing for. A list of all repositories is shown when you initally open it (yes, all, even if its in the 1000s). To filter repositories, a search bar is provided. Under the hood this search bar takes your query and runs a `repo:${yourQuery}` filter against codesearch. To speed up access to common repos, "favorites" are provided. Each repo has a star icon next to it, click it and a repo will be added to your favorites. Favorites are shown at the very top of the list. Thanks to Sean Renaud for the suggestion.

Branch/Tag/Commit Selector

This is really three selectors combined into one tabbed popup.
